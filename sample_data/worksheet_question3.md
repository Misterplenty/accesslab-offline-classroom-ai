# Worksheet: Python Loops

Question 1: What does a variable do?
A variable stores a value so the program can use it later.

Question 2: What does `print()` do?
`print()` shows text or numbers on the screen.

Question 3: What does `for item in numbers:` mean?
Short answer:
It means the program goes through the list one item at a time.

Longer answer:
- `numbers` is the whole list.
- `item` is the next value from the list.
- Each time the loop runs, `item` changes to the next value.

Example:
If `numbers = [3, 5, 8]`, then `item` becomes 3, then 5, then 8.
