# Hoja de bucles de Python

Un bucle `for` visita cada valor de una lista, uno por uno.

Ejemplo:

```python
numeros = [3, 5, 8]
for numero in numeros:
    print(numero)
```

En este ejemplo, `numero` vale 3, luego 5 y luego 8. `print(numero)` muestra el valor actual en la pantalla.

Frase clave: un bucle `for` ayuda a repetir la misma accion para cada elemento de una lista local.
